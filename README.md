# Quantum Tech ⚡

**Quantum Tech** is a responsive, modern e-commerce concept website designed for PC builders and tech enthusiasts. This project was developed as a CSS and JavaScript practice exercise to demonstrate proficiency in frontend design, interactive layout construction, and dynamic content rendering.

---

## 🚀 Overview

The website features a sleek, dark-themed UI tailored for high-performance computing aesthetics. It includes functional components such as dynamic product filtering, a search engine with suggestions, user authentication simulation, and a shopping cart/wishlist system.

> **Note:** This is an educational/demonstration project. It is not a real e-commerce platform; no actual purchases or transactions can be processed through this site.
> 
> 

---

## 🛠 Features

* **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens using modern CSS practices.


* **Dynamic Rendering:** Product lists and categories are rendered dynamically via JavaScript.


* **Search Functionality:** Real-time search suggestions with fuzzy matching to help users find specific components.


* **Interactive UI:**
* **User Authentication:** Simulated login/registration system using `localStorage`.


* **Cart & Wishlist:** Add, remove, and track items with persistent storage per user.


* **Product Modals:** Detailed views for every product with "Add to Cart" and "Wishlist" functionality.




* **Modern Styling:** Utilizes custom properties (`:root`), Flexbox, and CSS Grid for a robust, maintainable layout.



---

## 💻 Tech Stack

* **HTML5**
* **CSS3** (Custom properties, Flexbox, Grid)
* **Vanilla JavaScript** (ES6+)
* **Font Awesome** (Icons)
* **Google Fonts** (Poppins)
